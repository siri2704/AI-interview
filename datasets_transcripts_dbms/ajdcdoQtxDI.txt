so what is dbms that is database management system dbms is a software system that enables users to Define create maintain and control access to the database it provide an interface for users and application to store retrieve and manipulate data while ensuring the data integrity and security the database system has the following function it supports data management that is it helps to store retrieve and modify the data it provides transaction support this means it ensures that the modification to the database must either be successful or not done at all it provides the concurrency control which means it provide the simultaneous access of data to the users it Support Recovery it provide a recovery mechanism for data so that nothing is lost it also provide the security which means the data can be only accessed by the authorized users only it also supports data Integrity which means it maintains the accuracy of the data dbms supports data import export user management and many more
